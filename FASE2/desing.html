Sintaxis y Selectores CSS

Primero definiremos la sigla CSS = Cascading Style sheets, en español significa hojas de estilo en cascada. Esta sintaxis es basada en reglas y se puede utilizar para estilos muy básicos.

Este mecanismo da forma a documentos HTML y XML, las cuales son reglas simples para establecer como se va a mostrar un documento en la pantalla o como se va a imprimir.

COMO APLICAR CSS A UN DOCUMENTO HTML

Una hoja de estilos CSS no es que un archivo de texto, igual que un archivo HTML, solo que con la extensión .css, por tanto la forma de crearlas o editarlas es igual que usar HTML.
